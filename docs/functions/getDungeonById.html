    Function getDungeonById

    • getDungeonById(id: string): Dungeon | undefined

      Lookup a dungeon previously registered with an ID.

      Only dungeons that were created with an id option or had their id property set will be present in the registry. This function provides a global lookup mechanism for retrieving dungeon instances by their unique identifier, which is useful for serialization, room references, and cross-dungeon navigation.

      Parameters

      • id: string

        The dungeon id to look up

      Returns Dungeon | undefined

      The Dungeon instance or undefined when not found

      Example

      // Create a dungeon with an ID
      const dungeon = new Dungeon({
        id: "midgar",
        dimensions: { width: 10, height: 10, layers: 3 }
      });

      // Look it up later from anywhere
      const found = getDungeonById("midgar");
      console.log(found === dungeon); // true
