    Interface DungeonOptions

    Options for creating a Dungeon instance.

    Currently the only required option is dimensions, which describes the three-dimensional grid size for rooms. This type is intentionally simple so it can be extended in the future with additional configuration options.

    Example

    const options: DungeonOptions = {
      dimensions: { width: 10, height: 10, layers: 3 }
    };
    const dungeon = new Dungeon(options);
    interface DungeonOptions {
        dimensions: MapDimensions;
        id?: string;
    }
